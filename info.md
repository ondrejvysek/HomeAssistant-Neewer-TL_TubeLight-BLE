# Neewer Tube Light Integration for Home Assistant

Control your Neewer tube lights (TL60 RGB, TL120C) via ESPHome BLE proxy.

## Features

- ✅ Support for Neewer TL60 RGB
- ✅ Support for Neewer TL120C
- ✅ RGB color control
- ✅ Brightness control
- ✅ Color temperature control (TL120C)
- ✅ Easy setup via UI
- ✅ Works with ESPHome BLE proxy

## Requirements

- Home Assistant 2023.1 or newer
- ESP32 device with ESPHome configured as BLE proxy
- Neewer TL60 RGB or TL120C tube light

## Quick Start

1. Install this integration through HACS
2. Set up an ESP32 with ESPHome as a BLE proxy (see example configuration)
3. Add the Neewer Tube Light integration via UI
4. Enter your light's MAC address and model
5. Start controlling your light!
