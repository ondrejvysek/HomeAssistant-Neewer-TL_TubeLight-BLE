# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-29

### Added
- Initial release of Neewer Tube Light integration
- Support for Neewer TL60 RGB tube light
- Support for Neewer TL120C tube light
- RGB color control for both models
- Brightness control (0-100%)
- Color temperature control for TL120C (2700K-6500K)
- UI-based configuration flow
- Integration with ESPHome BLE proxy
- Comprehensive documentation
- HACS support
- Example ESPHome configuration
- English translations

### Features
- Easy setup through Home Assistant UI
- Device discovery via MAC address
- Full light entity support
- Compatible with all Home Assistant automations and scenes
- Works with ESPHome BLE proxy on ESP32 devices

### Documentation
- Installation guide
- Configuration instructions
- Troubleshooting section
- Usage examples
- ESPHome setup guide
